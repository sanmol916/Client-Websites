# Radiant Beauty — Bridal Makeup & Salon — Demo Website

A glamorous **bridal-luxe** single-page website for **Radiant Beauty – Bridal Makeup & Salon** in Meera Bagh, Paschim Vihar, New Delhi. Designed in a deep wine/maroon + rose-gold + ivory palette — distinct from the six previous salon sites (gold, red, blush, emerald, navy, violet).

## ✨ Highlights to show the client
- Elegant **Marcellus** serif + clean **Mukta** body
- Full-screen bridal hero with slow-zoom image, gold accents & scroll cue
- **"Looks We Create"** showcase (Bridal · Engagement · Reception · Party) — tailored to a makeup artist
- Artist about section with an arched, gold-framed portrait
- **Bridal Packages** section (Engagement Glow · Signature Bridal · Complete Wedding) with a featured "Most Loved" plan
- Portfolio-style gallery designed to highlight real bride photos
- Bride testimonials, booking form (with event date), live Google Map
- Click-to-call & WhatsApp + floating WhatsApp button
- Fully responsive + SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 320×120 transparent PNG |
| `hero.jpg` | Full-screen hero background | 1920×1080 (a bridal shot) |
| `about-1.jpg` | About / artist portrait | 800×960 |
| `portfolio-1.jpg` … `portfolio-6.jpg` | Portfolio grid | 900×900 |

## 📝 Quick edits
- **Phone:** search `9999672360` in `index.html`
- **Address / hours:** Booking + Footer sections of `index.html`
- **Packages / services:** Packages + Services sections of `index.html`
- **Colors:** edit `:root` variables in `styles.css` (try `--wine` and `--gold`)

## 🚀 Run it
Open `index.html` in any browser — no build step required.
