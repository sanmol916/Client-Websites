# Shemrock Salon — Demo Website

A sleek, modern **spa-luxe** single-page website for **Shemrock Salon** at Gole Market, Central Delhi. Designed in a deep emerald + cream + champagne-gold palette — distinct from the three previous salon sites (gold luxury, bold red, and blush boutique).

## ✨ Highlights to show the client
- Modern **Syne** display type + clean **DM Sans** body
- Hero with an arched image, floating rating badge & emerald accents
- Scrolling **service strip** in emerald + gold
- **Price-menu layout** for services (instead of cards) — feels like a real salon menu
- A unique **"Experience" steps section** (01–04) explaining how a visit works
- Asymmetric gallery, testimonials, booking form + live Google Map
- Click-to-call & WhatsApp + floating WhatsApp button
- Fully responsive + SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 320×120 transparent PNG |
| `hero.jpg` | Hero arch image | 1000×1180 |
| `about-1.jpg` | About main image | 800×900 |
| `about-2.jpg` | About small overlapping image | 600×460 |
| `gallery-1.jpg` … `gallery-6.jpg` | Gallery grid | 900×900 |

## 📝 Quick edits

## 🚀 Run it
