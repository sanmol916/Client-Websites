# Sadhna Beauty Salon · Academy & Studio — Demo Website

A **soft, modern boutique-style** single-page website for **Sadhna Beauty Salon, Academy & Studio** in New Ashok Nagar, Delhi. Designed in a blush / rose / deep-plum palette — a completely different look from the gold SMUK site and the bold red Makeover site.

## ✨ Highlights to show the client
- Elegant **Fraunces** serif + clean **Poppins** body type
- Soft gradient hero with **floating chips**, an arch-shaped image & decorative blobs
- Floating **highlights card** (Salon · Academy · Studio · Hours)
- Organic, rounded shapes throughout (no two corners alike)
- **Dedicated Academy / Courses section** — unique to this dual salon + academy business
- Salon services grid, studio gallery, client & student testimonials
- Smart **booking form** that lets visitors pick a salon service **or** an academy course
- Live Google Map + click-to-call & WhatsApp + floating WhatsApp button
- Fully responsive + SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 340×120 transparent PNG |
| `hero.jpg` | Hero arch image | 1000×1150 |
| `about-1.jpg` | About main image | 800×900 |
| `about-2.jpg` | About small overlapping image | 600×500 |
| `gallery-1.jpg` … `gallery-5.jpg` | Studio gallery | 900×900 |
