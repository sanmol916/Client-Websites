Media) using the **exact same filename**. No code changes needed.

## How to replace

### Option A — Static site
Drop the new file into the `images/` folder with the **same name**, overwriting
the placeholder. Refresh the page.

### Option B — WordPress
Upload each photo to **Media Library** using the same filename. If you convert
these pages into WordPress, point the image blocks to the uploaded media.

## Regenerating placeholders
The placeholder images were generated by `make_placeholders.py`. You can delete
this script and the placeholders once real photos are in place.
