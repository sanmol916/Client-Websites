# SMUK Salon — Demo Website

A premium, fully responsive single-page website built for **SMUK Salon**, Neb Sarai, New Delhi.

## ✨ Highlights to show the client
- Luxury salon aesthetic (charcoal + champagne gold) with elegant serif typography
- Animated hero with stat counters, slow-zoom background & scroll cue
- Sticky glass navigation + mobile slide-in menu
- Scroll-reveal animations throughout
- Services, gallery, pricing, reviews sections
- Booking form + live Google Map + click-to-call & WhatsApp buttons
- Floating WhatsApp chat button
- SEO meta tags + social share tags
- 100% responsive (desktop, tablet, mobile)

## 🖼️ How to add the client's images
All images live in the `assets/` folder. Just drop in files with these exact names and they appear automatically — no code changes needed.

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text if missing) | 300×120 (transparent PNG) |
| `hero.jpg` | Full-screen hero background | 1920×1080 |
| `about-1.jpg` | About main photo | 800×960 |
| `about-2.jpg` | About floating photo | 600×400 |
| `gallery-1.jpg` … `gallery-6.jpg` | Gallery grid | 800×800 |

Until real images are added, each spot shows a labeled placeholder so the layout stays intact.

## 📝 Quick edits
- **Phone number:** search for `7301020300` in `index.html`
- **Address / hours:** in the Booking and Footer sections of `index.html`
- **Prices & services:** in the Pricing and Services sections of `index.html`
- **Colors:** edit the `:root` variables at the top of `styles.css`

## 🚀 Run it
Just open `index.html` in any browser — no build step required.
