# Hair Masters Salon Pitampura — Demo Website

A modern **urban-premium** single-page website for **Hair Masters Salon, Pitampura** in Engineers Enclave, Harsh Vihar, New Delhi. Designed in a midnight-navy + copper palette — distinct from the four previous salon sites (gold luxury, bold red, blush boutique, emerald spa).

## ✨ Highlights to show the client
- Modern **Space Grotesk** display type + clean **Inter** body
- Split hero with a copper-accented image, floating "EST. Pitampura" chip & stats row
- Navy + copper scrolling **marquee** strip
- Numbered **services grid** with copper underline-on-hover
- **"Meet the Masters" team section** — showcases stylists with photo placeholders (unique to this site & perfect for the brand name)
- Asymmetric gallery, testimonials, booking form + live Google Map
- Click-to-call & WhatsApp + floating WhatsApp button
- Fully responsive + SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 320×120 transparent PNG |
| `hero.jpg` | Hero image | 1000×1150 |
| `about-1.jpg` | About image | 900×950 |
| `team-1.jpg` … `team-4.jpg` | Master / stylist photos | 700×800 |
| `gallery-1.jpg` … `gallery-6.jpg` | Gallery grid | 900×900 |
