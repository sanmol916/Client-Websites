# Amit Beauty Parlour — Demo Website

A warm, friendly **modern** single-page website for **Amit Beauty Parlour** near New Delhi Railway Station, Aram Bagh, Paharganj. Designed in a royal-violet + warm-amber + lilac palette — distinct from the five previous salon sites (gold, red, blush, emerald, navy).

## ✨ Highlights to show the client
- Friendly **Outfit** display type + clean **Mulish** body
- Soft violet-gradient hero with floating rating & "Open Today" cards
- Floating **trust strip** (Expert · Affordable · Quality · Easy to reach)
- Rounded, approachable service cards with amber hover accents
- **Special Offers / combo-deal section** with featured "Best Value" package — unique to this site
- Asymmetric gallery, client testimonials, booking form + live Google Map
- Click-to-call & WhatsApp + floating WhatsApp button
- Fully responsive + SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 320×120 transparent PNG |
| `hero.jpg` | Hero image | 1000×1130 |
| `about-1.jpg` | About main image | 800×900 |
| `about-2.jpg` | About small overlapping image | 600×480 |
| `gallery-1.jpg` … `gallery-5.jpg` | Gallery grid | 900×900 |
