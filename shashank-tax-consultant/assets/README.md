# 📁 Images folder — how to add your own

This folder already contains **labeled placeholder images**. To use your own,
just **replace each file with your image using the EXACT SAME file name**.
The website picks them up automatically — no code changes needed.

## Files to replace

| File name        | Where it appears on the site        | Best size (px)      | Format |
|------------------|-------------------------------------|---------------------|--------|
| `logo.png`       | Header + footer logo (square badge) | 240 × 240 (square)  | PNG (transparent) |
| `hero.jpg`       | Big image on the home banner        | 800 × 900 (portrait)| JPG |
| `shashank.jpg`   | About-section portrait photo        | 700 × 800 (portrait)| JPG |
| `client-1.jpg`   | 1st testimonial + hero avatar       | 200 × 200 (square)  | JPG |
| `client-2.jpg`   | 2nd testimonial + hero avatar       | 200 × 200 (square)  | JPG |
| `client-3.jpg`   | 3rd testimonial + hero avatar       | 200 × 200 (square)  | JPG |
| `favicon.svg`    | Small icon in the browser tab       | 64 × 64             | SVG/PNG |

## ⚠️ Important rules
1. **Keep the exact same file name** (e.g. your photo must be saved as `hero.jpg`, not `hero.png` or `Hero.JPG`).
2. Keep roughly the same shape (portrait vs square) so images aren't stretched.
3. `logo.png` should be a **square emblem/icon** — the firm name already appears as text next to it.
4. If you only have a photo in a different format, either rename it to match, or tell the developer and the link will be updated.

## Tip
If you don't have client photos for the testimonials, you can leave
`client-1/2/3.jpg` as they are, or replace them with a neutral image.
