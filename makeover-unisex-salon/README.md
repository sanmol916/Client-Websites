# Makeover Unisex Salon — Demo Website

A **bold, organized** single-page website for **Makeover Unisex Salon**, Arjun Nagar, Safdarjung Enclave, New Delhi. Designed in a high-impact red / black / white style (distinct from the elegant gold SMUK Salon site).

## ✨ Highlights to show the client
- Bold editorial design with **Anton** display type + clean **Archivo** body font
- Split hero with offset image frame, red shadow & rotated "EST. DELHI" sticker
- Scrolling service **ticker** strip
- Sharp **grid-based services** (numbered 01–08) with hover invert
- **Why Us / Amenities** band using the real Google data: restrooms, appointments, card & NFC payments, kid-friendly
- Bold **pricing plans** with hard-shadow hover
- Asymmetric **lookbook gallery**
- **4.9★ / 501 reviews** featured prominently
- Booking form + live Google Map + click-to-call & WhatsApp
- Floating WhatsApp button, fully responsive, SEO meta tags

## 🖼️ How to add the client's images
Drop files into the `assets/` folder using these exact names — no code changes needed:

| File | Where it shows | Suggested size |
|------|----------------|----------------|
| `logo.png` | Header logo (falls back to text) | 320×120 transparent PNG |
| `hero.jpg` | Hero portrait frame | 1000×1250 |
| `about-1.jpg` | About main image | 800×800 |
| `about-2.jpg` | About small overlapping image | 600×400 |
| `gallery-1.jpg` … `gallery-5.jpg` | Lookbook grid | 800×800 |

## 📝 Quick edits
