## 🎯 How to Deploy to WordPress

### Option 1: Quick Deployment (Recommended)

1. **Download the ZIP file**
   - File: `amar-legal-wordpress-theme.zip`
   - Location: In your repository root

2. **Upload to WordPress**
   ```
   WordPress Admin → Appearance → Themes → Add New → Upload Theme
   ```
   - Click "Choose File"
   - Select `amar-legal-wordpress-theme.zip`
   - Click "Install Now"
   - Click "Activate"

3. **Configure the theme**
   - Follow the steps in **QUICKSTART.md** (15 minutes)
   - Or follow **INSTALLATION.md** for detailed instructions

### Option 2: Manual Installation via FTP

1. **Extract the ZIP file**
   - Extract `amar-legal-wordpress-theme` folder

2. **Upload via FTP**
   - Connect to your server
   - Navigate to: `/wp-content/themes/`
   - Upload the entire `amar-legal-wordpress-theme` folder
   - Go to WordPress Admin → Appearance → Themes
   - Activate the theme

## ⚙️ Initial Setup (5 Steps)

After activating the theme:

### Step 1: Set Homepage
```
Settings → Reading
- Select "A static page"
- Homepage: Create and select "Home" page
- Posts page: Create and select "Blog" page
- Save Changes
```

### Step 2: Create Menu
```
Appearance → Menus
- Create "Main Menu"
- Add pages: Home, About, Services, Contact
- Assign to "Primary Menu" location
- Save Menu
```

### Step 3: Add Contact Information
```
Appearance → Customize → Contact Information
- Phone Number
- Email Address
- Office Address
- Office Hours
- Publish
```

### Step 4: Upload Logo
```
Appearance → Customize → Site Identity
- Upload Logo (200×80px PNG recommended)
- Add Site Icon (favicon)
- Publish
```

### Step 5: Customize Hero Section
```
Appearance → Customize → Theme Options
- Hero Title: "Amar Legal"
- Hero Description: Your tagline
- Publish
```
