# ⚡ Quick Start Guide - Amar Legal WordPress Theme

Get your website running in **15 minutes**!

---

## 🚀 Super Fast Setup (5 Steps)

### STEP 1: Upload Theme (3 minutes)
1. Create a ZIP file of the `amar-legal-wordpress-theme` folder
2. Go to **WordPress Admin → Appearance → Themes → Add New → Upload Theme**
3. Upload ZIP and click **Activate**

### STEP 2: Set Homepage (2 minutes)
1. Go to **Pages → Add New**
2. Title: "Home", leave content blank, click **Publish**
3. Go to **Settings → Reading**
4. Select "A static page" and choose "Home"
5. Click **Save Changes**

### STEP 3: Create Menu (3 minutes)
1. Go to **Appearance → Menus**
2. Create "Main Menu"
3. Add pages: Home, About, Services, Contact
4. Check "Primary Menu" location
5. Click **Save Menu**

### STEP 4: Add Your Info (5 minutes)
1. Go to **Appearance → Customize → Contact Information**
2. Add:
   - Phone number
   - Email address
   - Office address
   - Office hours
3. Click **Publish**

### STEP 5: Upload Logo (2 minutes)
1. In Customizer, go to **Site Identity**
2. Upload your logo (200×80px PNG recommended)
3. Click **Publish**

## 💡 Quick Tips

### Change Colors
Edit `style.css` line 23-28:
```css
--primary-color: #1a3c5e;    /* Navy blue */
--secondary-color: #c9a961;  /* Gold */
```

### Update Hero Text
**Appearance → Customize → Theme Options**
- Hero Title
- Hero Description

### Add Social Links
**Appearance → Customize → Social Media Links**
- Facebook, Twitter, LinkedIn, Instagram
